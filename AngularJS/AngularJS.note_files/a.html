              Lungo 我还不知道是什么东西。刚才了解了一下，是一个针对移动端的HTML5框架吧。这方面我只了解过 jQuery Mobile 和 dojo 的移动支持部分。一般来说，考虑结合的话：

              1. 同时载入它们没有大冲突，并且都能正常工作，这是前提。
              2. ng 的 bootstrap 需要在 DOM 结构定下来之后，即你可能需要控制其它框架的初始化过程。
              3. 数据层面的工具随便用，这个没影响。
              4. 涉及 DOM 结构的功能，可能需要做在自定义的 ng 的命令中。

              近期我会尝试一下 Lungo ，到时再补充吧。
