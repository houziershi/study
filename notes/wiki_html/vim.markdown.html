makrdown转html

markdown.pl

perl官方提供了使用 Markdown.pl 文件进行转换。 

$ unzip Markdown_1.0.1.zip
$ sudo cp Markdown_1.0.1/Markdown.pl /usr/local/bin/

将markdown.text文本转成markdown.html文件： 

/path/to/Markdown.pl --html4tags markdown.text > markdown.html

结合了vim修改.vimrc： 

let mapleader = ","
nnoremap <leader>md :%!/usr/local/bin/Markdown.pl --html4tags <CR>

此时在VIM普通模式下输入,md就可以将markdown文本转换为html格式。如果不想通过VIM 。

showdown.js

showdown.js源码中提供了例子： 

var Showdown = require('showdown');
var converter = new Showdown.converter();

converter.makeHtml('#hello markdown!');

// <h1 id="hellomarkdown">hello, markdown</h1>
