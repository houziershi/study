文件

写入

import java.io._

val writer = new PrintWriter(new File("tmp.txt'))
writer write "test"
writer.close()

读取

import scala.io.Source

Source.fromFile("tmp.txt").foreach { print }

foreach()每次读一个字符（输入有缓冲，不用担心性能）。用getLines()可以读一行 。getLine(index)可以取得指定行，是索引，从0开始。

读取URL

import scala.io.Source
import java.net.URL

val source = Source.fromURL(new URL("http://ifconfig.me/ip"))
source.getLines().foreach{ println }
